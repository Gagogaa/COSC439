# Before running the program
First check the python version:
```
>$ python --version
Python 2.7.14
```

If the version is less than "3.6.0" try:
```
>$ python3 --version
python3: command not found...
```

If the command does not exist please install python 3 from:
https://www.python.org/downloads/release/python-364/

# Running the program
(Assuming python 3 is accessable via the python3 command)

```
>$ python3 gmann1_rmatrix.py
```

Using a custom input file with the "-i" argument:
```
>$ python3 gmann1_rmatrix.py -i gmann1_input.txt
```

Renaming the output file with the "-o" argument:
```
>$ python3 gmann1_rmatrix.py -o gmann1_output.txt
```

Using a custom input and output filenames:
```
>$ python3 gmann1_rmatrix.py -i gmann1_input.txt -o gmann1_output.txt
```
